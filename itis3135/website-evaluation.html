Website Evaluation

Cloudflare.com

This is the homepage of Cloudflare a CDN and DDOS protection service.

  • Contrast - The colors constract with a light background and dark text or a dark background and light text making this website easy to read.
  • Repetition - Each part of the website as your scroll down uses vector images acompanied with text showing reptition.
  • Alignment - All of the elements on the webpage are centered and are aligned in a vertical order. No element was found out of place.
  • Proximity - The website is clearly divided into differenet section using different background colors with related elements using the same background color.

Pnwx.com

This is the website of Pacific Northwest X-ray selling various X-ray related equipment.

  • Contrast - The lack of contrast with the dark background combined with dark text makes it difficult to read. Modifying the colors with either light on dark or dark on light would greatly improve this.
  • Repetition - Each of the product pages uses a different layout lacking any repetition. Using the same layout of each product would improve this.
  • Alignment - There is some basic alignment with different products falling under differnet headers but it can be improved with more consistent product layouts.
  • Proximity - The website does a fair job with proximity as it is easy to identify different parts of the website but can be improved with better placement of product images.
